Charles配置手机代理

2019-07-17

技术 前端

前情提要

因为换了OS系统，之前的抓包工具Fiddler没法用了（虽然有OS版本的但是比较麻烦且做得不太好）所以必须要尝试新的抓包工具了。同时考虑到工作上移动端H5页面占比很大，所以需要在手机上抓包调试的情况非常多，因而抓包工具也必须要兼顾到此需求。在网上搜索一番，比对过广大网友的推荐后，选择了Charles~

虽然一开始用起来不太顺手，但是多用几次之后感觉还是很不错的，也推荐给大家~
(废话 (╯‵□′)╯︵┻━┻ 不是用的Charles谁点开这篇文章啊！)

好好好，别急别急，马上就进入正题了~

配置Charles

  1. 打开Charles。
  2. 在顶部菜单栏中，选中Proxy -> Proxy Settings。
  3. 在Proxies中，配置port或者默认使用8888，然后⚠️勾选Enable transparent HTTP proxing。

配置手机

  1. 将手机连接到和电脑同一个网络中。
  2. 进入到你手机的wifi配置里面，找到配置代理的地方。如果不知道怎么弄可以网上搜索 机型+配置代理。
  3. 服务器填写你的电脑IP地址，port填写你之前填写的或者默认为8888。
    (电脑ip可以在命令行中查看，Win系统为ipconfig | Os系统为ifconfig；还可以通过Charles查看：Help -> Local IP Address)

注意事项

⚠️⚠️⚠️电脑一定要用 网线 连接网络，绝对不能用wifi，否则就会像我把上面所有的准备都做好了，但就是出现不了弹窗。在网上搜索半天，然后把网线一插，啊哈？？？居然就可以了…🤷‍
