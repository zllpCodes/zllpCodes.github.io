一页A4纸打印两页内容

2019-06-12

读书

前情提要

看书是个刚需，但是不喜欢看电子书咋办？买纸质的呗。
一手书不贵，但是书看完了堆着搬家的时候就知道了，欲哭无泪啊。那就卖书呗。
卖二手书不划算，平台上2.3折的卖，心疼啊。
要是能把想看的书看了，想收藏的再买正版的就好了～
所以尝试了一下：网上搜索PDF版本，然后打印出来，一页打印两页内容，双面打印。这样肯定比买新书便宜说不定比买二手的还划算。还有个好处：要是借给别人看，书弄坏了不心疼～

打开PDF

在chrome中打开PDF文件。可以是你本地的PDF文件，也可以是在网上直接找到的PDF文件，总之，在chrome中打开它。

如果你问我chrome是什么，答应我，一定要去下载chrome用一下好吗？⬇️
戳我戳我(●’◡’●)

唤起Printer配置

Mac OS系统下

cmd + p

Win系统下

ctrl + p

然后点击 

使用系统对话框进行打印...

配置操作

  1. 打印机。选择你要使用的打印机就好了。
  2. 预置。选择黑白或者彩色。
  3. 页数。选择你要打印的页数，全部或者某个页码到某个页码
  4. 纸张大小。默认选择A4。
  5. 打印方向。纵向。
  6. ⚠️关键操作来了！点击介质和质量的选项框，选到布局一项。
  7. ⚠️在新出现的子配置项中，每张页数选择2。
  8. ⚠️在双面打印中，选择短边装订。这个会影响翻页的时候是根据短边翻页还是长边。现在这样一页打印两页内容就应该是从短边翻页才合适。
